## Usage

```js
const ISO6391 = require('./')
console.log(ISO6391)
// {
//   ab: 'Abkhazian',
//   aa: 'Afar',
//   af: 'Afrikaans',
//   ak: 'Akan',
//   sq: 'Albanian',
//   am: 'Amharic',
//   ar: 'Arabic',
//   an: 'Aragonese',
//   hy: 'Armenian',
//   as: 'Assamese',
//   av: 'Avaric',
//   ae: 'Avestan', ...
```

## Install

This project uses [node](https://nodejs.org) and [npm](https://www.npmjs.com).

```sh
$ npm install ISO639-1-tags
$ # OR
$ yarn add ISO639-1-tags
```
